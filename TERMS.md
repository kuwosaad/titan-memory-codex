# Terms of Service

Titan Memory for Codex is provided as a local developer tool for adding durable memory, recall, graph inspection, pattern workflows, and passive capture to Codex.

## Your responsibility

You are responsible for choosing whether to enable MCP tools and whether to trust hooks in Codex. Do not store secrets, regulated data, or data you do not have permission to process.

## Local runtime

The plugin depends on a local Titan runtime or package launcher. You are responsible for your local Titan configuration, provider credentials, and any network calls made by that runtime.

## No warranty

This plugin is provided as-is, without warranty. Verify outputs before relying on them for production, legal, medical, financial, or safety-critical decisions.

## Data removal

Titan stores Codex data locally under `~/.titan/agents/codex` by default. You can remove local data by deleting that directory.
